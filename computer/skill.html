# 电脑技巧

# 1、批量重命名文件

操作方法： 选中几个需要重命名的文件 按 F2 键 → 重命名这些文件中的一个，接着所有被选中的文件将会被重命名为新的文件名并依次在末尾处加上递增的数字。

# 2、快捷复制文件路径

按住 Shift 选中文件，右键-> 复制为路径

# 3、输入特殊字符

Win+R 输入 charmap 就可以调出字符映射表，然后找到自己需要的符号，选择复制就可以啦~

# 4、快速预览文件内容

在某一文件夹下，按 Alt+P 可以实现隐藏或者显示资源管理器的预览窗口，对文件进行预览。

# 5、讲述人功能，让你“听”屏幕内容

“Win 键 + 回车”组合键可以开启“讲述人”功能，开启“讲述人”后，我们可以用鼠标、Tab 键或者触控选择屏幕上各种元素，系统会详尽读出当前元素的所有文字与符号内容。 退出讲述人功能，按“大写锁定键＋ Esc”即可。

# 6、DOS 命令查看局域网所有 IP

查看局域网内所有 IP ：arp -a

查看局域网内所有主机名： net view

查看 IP 地址对应的主机名和 MAC 地址： nststat -a 192.168.0.101

# 7、打开电脑组策略的 3 种方式

1.Win+R 输入 gpedit.msc

2.左下搜索框：输入 编辑组

3.C:\WINDOWS\system32\gpedit.msc

# 8、cmd 修改 UTF-8 编码

1.输入命令CHCP 65001

2.鼠标放在命令窗口的标题部分右键，在弹出的右键菜单上面选择【属性】,在新弹出的窗口选择【字体】选项卡 然后在下面的字体里选择【Lucida Console】这个字体
